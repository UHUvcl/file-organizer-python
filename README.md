# File Organizer (Python)

A simple Python script to organize files in a folder by type.

## Features
- Automatically creates folders (Images, Videos, Audio, Documents, Archives, Others)
- Moves files based on extension
- Avoids overwriting by renaming duplicates

## How to run
1. Run the script:
   python file_organizer.py
2. Paste the folder path you want to organize (e.g. Downloads)
